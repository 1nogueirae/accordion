# Accordion

A simple accordion component built with HTML, CSS, and JavaScript.

## Description

This project demonstrates a basic accordion interface where clicking on a card expands its content while automatically collapsing other cards. The accordion is styled using Bootstrap and custom CSS.

## Features

- Expand/collapse cards on click
- Only one card can be open at a time
- Visual indicators (arrows) show card state
- Responsive design with Bootstrap

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5.3.8

## Usage

Simply open `index.html` in your browser. Click on any card to expand and view its content. Clicking another card will collapse the previously opened card.
